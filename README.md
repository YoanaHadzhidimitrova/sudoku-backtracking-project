# Sudoku Solver using Backtracking

Final project for the course **Math Concepts for Developers**.

## Project Description

This project explores the mathematical structure of Sudoku puzzles and demonstrates how they can be solved using a backtracking algorithm.

The notebook explains:
- the combinatorial nature of Sudoku
- constraint satisfaction problems
- the backtracking search algorithm
- a Python implementation of a Sudoku solver

The goal is to show how a recreational puzzle can be analyzed using mathematical reasoning and algorithmic thinking.
